# my-pocket-webtoon-admin
[my-pocket-webtoon](https://github.com/GuardJo/my-pocket-webtoon) 서비스에 대한 어드민 서비스

> `my-pocket-webtoon`  내 작품 및 회원 관리를 위한 백오피스
>

# 주요 기능

## 1. 작품 관리 `필수`

1. 작품 업로드 기능
2. 작품 삭제 기능

## 2. 회원 관리 `필수`

1. 서비스 접근 가능한 회원 추가
2. 회원 별 작품 조회 이력 관리
3. 회원 탈퇴 처리

## 3. 매트릭 관리

1. FO 모듈 서버 상태 현황 모니터링
2. 방문자 기록 확인
